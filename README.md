# 🐦 Flappy Bird

A recreation of the classic **Flappy Bird** game. Flappy Bird is a simple arcade game where the player taps to make a bird fly through pipes, each successful pass adding a point to the player's total score. The goal is to survive as long as possible.

## 🎮 Features

- Smooth bird movement and gravity physics
- Procedurally generated pipes
- Collision detection
- Real-time score tracking
- Game over and restart system
- Simple and responsive controls

## 🕹️ Controls

- **Enter** or **Return** → Start
- **Space** → Flap
- **Esc** → Exit 

## ▶️ How to Run

1. Install [LÖVE2D](https://love2d.org/)
2. Clone this repository:
	```bash
	git clone https://github.com/Aquanouna/Flappy-Bird.git flappy
	```
3. Navigate to the folder:
	```bash
	cd flappy
	```
4. Run the game:
	```bash
	love .
	```
